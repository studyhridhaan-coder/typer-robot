**Software Used**

**1. Inkscape**

* Draws simple line designs  
* Exports files in SVG format  
 
It is free, simple, and commonly used for pen plotter projects.

**2. Inkscape G-code Extension**
Used to convert drawings into machine instructions
* Converts SVG drawings into G-code  
* Allows basic control of speed and size  
  
It directly connects drawing creation to machine movement.

**3. Universal G-code Sender (UGS)**
Used to send instructions to the plotter
* Connects to the plotter via USB  
* Starts, pauses, and stops the drawing process  
* Shows a preview of the motion path  

It is easy to use and widely supported.


**4. Arduino IDE**
Used to upload and test code on the controller
* Uploads calibration and test programs  
* Used during setup and debugging  
It is the standard tool for programming Arduino-based controllers.


 **5. GRBL Firmware**
Runs on the controller inside the plotter
* Interprets G-code instructions  
* Controls motor movement and pen action  

It is lightweight, reliable, and designed for simple CNC machines.
